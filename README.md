# R code for the continumm central classifier (CCC)
## Instruction
- Tested at RStudio. 
- Run R script **main.R**. 
- R script **functions.R** contains all the supporting functions.
- Saved R image-files and figures go into **Rimage** and **figure**, respectively.

## Reference 
**Zhou, Z.**, & Sang, P. (2022). Continuum centroid classifier for functional data. _Canadian Journal of Statistics_, 50, 200-220.
[doi:10.1002/cjs.11624](https://onlinelibrary.wiley.com/doi/abs/10.1002/cjs.11624)
